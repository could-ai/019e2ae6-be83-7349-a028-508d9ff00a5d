# Glass Manufacturing in the Middle East Study

A responsive, beautiful Flutter application presenting a short study on the glass manufacturing industry across Egypt, Saudi Arabia (KSA), Kuwait, UAE, and Qatar.

## Features
- **Responsive Layout:** Adaptive card grids that look great on mobile, tablet, desktop, and web views.
- **Categorized Sections:** Data beautifully grouped by country.
- **Modern UI:** Material 3 implementation with semantic coloring and polished, readable typography.
- **Dark/Light Mode:** Automatically respects the system theme for comfortable reading.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **Design:** Material 3

## Setup & Run
1. Ensure you have Flutter installed.
2. Clone the repository and navigate into it.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the application on your preferred device or browser.
